# Math Tutor — T192/T193

Engineering Mathematics practice app for Open University T192/T193 courses.

## Features
- 15 topic areas (BIDMAS → Integration)
- Mastery tracking (10 correct @ 80% accuracy)
- 3 difficulty levels
- Hints & step-by-step solutions
- Progress saved locally
- Works offline (PWA)
- Dark theme
